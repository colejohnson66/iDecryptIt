Using iDecryptIt

In order to use iDecryptIt, you need to have a firmware and a key. To get the key for a certain version, you click on the appropriate device and version. The keys for the device and version you chose will be displayed at the bottom of the window. Simply copy and paste the key into the "Key" text box in the decryption area. Select an input and output file and click "Decrypt". A progress bar will appear showing the progress of decryption. When it disappears, your file will have finished decrypted.

You can also extract the file you just decrypted. Navigate to the extraction area. Select an input file and output location and click "Extract". A window full of text will appear listing the files extracted. When it disappears, your firmware will have finished extracting.
