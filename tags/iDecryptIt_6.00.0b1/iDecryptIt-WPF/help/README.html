DOWNLOADING and INSTALLING

iDecryptIt currently is Windows only and is available in three versions: an Installer EXE, a ZIP archive, and an SVN to track the changes.

If you downloaded the Installer, just run it and it should install to "C:/Program Files/Hexware/iDecryptIt/" or the directory you chose. To run it, use the installed icon on the desktop, through the Start Menu, or the Quick Launch if you installed an icon there.

If you downloaded the ZIP archive, you can just extract them to "C:/Program Files/Hexware/iDecryptIt" or any directory you choose.

If you pulled from the SVN, it will work also, but you need to compile it with Visual C# Express or better and run {SVNROOT}/trunk/iDecryptIt-WPF/bin/Debug/iDecryptIt.exe
To use, just query the HEAD and navigate to the corresponding directory below.

USAGE

In order to use iDecryptIt, you go to the device area and select the device. The keys will display below in the tab area. Copy and paste the key into the "Key: " text box on in the decrypt area. Select an input and an output location and click "Decrypt". iDecryptIt will decrypt the File-Vault encrypted DMG. When it is done decrypting, you will get an alert saying that it is done decrypting.
